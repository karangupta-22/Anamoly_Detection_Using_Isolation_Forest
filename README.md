# Anomaly Detection using Isolation Forest

## Overview
This project implements an unsupervised machine learning approach for anomaly detection using the Isolation Forest algorithm. The model identifies outliers in high-dimensional datasets by isolating anomalous observations, making it effective for applications such as fraud detection, network security, and data quality monitoring.

## Features
- Anomaly detection using Isolation Forest
- Data preprocessing and cleaning
- Feature scaling and transformation
- Performance evaluation using AUC-ROC, Precision, Recall, and F1-Score
- Hyperparameter tuning for improved detection accuracy
- Data visualization and exploratory analysis

## Tech Stack

## Dataset
The project can be applied to any tabular dataset containing numerical or categorical features. Data preprocessing includes:
- Handling missing values
- Feature encoding
- Feature scaling
- Outlier analysis

## Model Evaluation
The model performance is assessed using:
- AUC-ROC Score
- Precision
- Recall
- F1-Score
